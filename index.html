CUDACadabra

One‑click installer for the NVIDIA CUDA ecosystem on Linux.

What it does

  • Detects your GPU and driver version.
  • Installs matching CUDA toolkit, cuDNN, and related libraries.
  • Configures environment variables and verifies the installation.

Quick start

git clone https://github.com//CUDACadabra.git
cd CUDACadabra
chmod +x install.sh
./install.sh

For detailed options, see the README.
